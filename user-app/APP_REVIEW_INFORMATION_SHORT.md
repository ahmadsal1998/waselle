**App Overview:**
Awsaltak is a delivery service app that connects customers with delivery drivers. Users can place delivery orders, track drivers in real-time on a map, and view order history.

**For Apple Reviewers - Review Mode Active:**
Review Mode is automatically active on TestFlight builds. No login required - test user account is auto-authenticated. Mock orders and test drivers are preloaded. Default location is set to Ramallah center. You can immediately test all features without setup.

**Normal User Flow:**
Users register with email/phone and password (OTP verification may apply). They can save multiple addresses, place delivery orders selecting saved or new addresses, track drivers in real-time on the map, and view order history.

**Review Mode Details:**
Review Mode allows full testing without a separate driver app. Mock orders and test drivers are provided. Review Mode only activates on TestFlight and is automatically disabled in App Store releases.

**Special Notes:**
- No separate driver app needed for testing
- All test data is preloaded, no login required
- App demonstrates all main functionalities: placing orders, tracking, viewing history
- Review Mode works automatically - no configuration needed
